# Response Templates

Fill-in templates for common peer review responses.

## Agreement Templates

### Simple Agreement
```
We thank the reviewer for this suggestion. We have [specific action]
as recommended.

Changes made: Page X, lines Y-Z.
```

### Agreement with Enhancement
```
We appreciate this excellent suggestion. We have not only [basic action]
but also [additional improvement]. We believe this strengthens the
manuscript.

Changes made: Page X, lines Y-Z; Page A, lines B-C.
```

### Agreement That Required Substantial Work
```
We thank the reviewer for identifying this important issue. Addressing
this concern required [describe effort: new analysis, additional data,
substantial rewriting]. We have now [summary of changes].

Changes made: [List all locations with page/line numbers]
```

---

## Disagreement Templates

### Respectful Disagreement with Evidence
```
We thank the reviewer for this thoughtful comment. We respectfully
maintain our original interpretation because [reason 1] and [reason 2].
[Cite supporting evidence if available.]

However, we recognize that readers may share the reviewer's perspective.
To address this, we have added text clarifying [specific clarification].

Changes made: Page X, lines Y-Z.
```

### Disagreement with Compromise
```
We appreciate the reviewer raising this concern. While we believe
[our position] because [reasoning], we acknowledge that [reviewer's
valid point]. As a compromise, we have [action taken].

Changes made: Page X, lines Y-Z.
```

### Disagreement Due to Misunderstanding
```
We thank the reviewer for this comment, which highlighted that our
original text was unclear. We believe there may have been a
misunderstanding: [brief clarification]. We have revised the text
to make this clearer.

Changes made: Page X, lines Y-Z.
```

## Cannot-Address Templates

### Resource/Data Limitations
```
We appreciate this suggestion. Unfortunately, [specific reason:
original samples are no longer available / this data was not collected /
we lack access to required equipment]. We have acknowledged this as
a limitation in the revised Discussion.

Changes made: Page X, lines Y-Z (limitation added).
```

### Beyond Scope
```
We thank the reviewer for this interesting suggestion. While this
analysis would certainly be valuable, it is beyond the scope of the
current study because [reason: would require different study design /
addresses different research question / requires data we cannot obtain].
We have added this as a direction for future research.

Changes made: Page X, lines Y-Z (future direction added).
```

### Would Fundamentally Change Study
```
We appreciate this suggestion. However, implementing this change would
fundamentally alter [aspect of study], which we believe would [negative
consequence]. We have instead [alternative action] to address the
underlying concern.

Changes made: Page X, lines Y-Z.
```

### Alternative Offered
```
We thank the reviewer for this suggestion. While we are unable to
[requested action] because [reason], we have [alternative action]
as a partial solution. We hope this addresses the reviewer's concern.

Changes made: Page X, lines Y-Z.
```

## Clarification Templates

### Simple Clarification
```
We thank the reviewer for noting this ambiguity. We have revised the
text to clarify that [clarification].

Changes made: Page X, lines Y-Z.
```

### Clarification with Additional Information
```
We appreciate this question. To clarify: [explanation]. We have added
text to the manuscript explaining this more clearly, and have also
included [additional detail] for completeness.

Changes made: Page X, lines Y-Z.
```

### Clarification Requiring New Section/Paragraph
```
We thank the reviewer for identifying this gap. The original manuscript
did not adequately explain [topic]. We have added a new [paragraph/
subsection] that describes [content].

Changes made: Page X, lines Y-Z (new [paragraph/subsection] added).
```

## Additional Analysis Templates

### Analysis Performed, Supports Original Conclusion
```
We thank the reviewer for this suggestion. We have performed the
requested analysis. [Describe analysis and results.] These results
support our original conclusion that [conclusion].

Changes made: Page X, lines Y-Z (results added); Figure/Table N
[if applicable].
```

### Analysis Performed, Modifies Conclusion
```
We thank the reviewer for this important suggestion. We have performed
the requested analysis, which revealed [finding]. Based on these
results, we have modified our conclusion to [revised conclusion].
We are grateful to the reviewer for strengthening our analysis.

Changes made: Page X, lines Y-Z (results); Page A, lines B-C
(revised conclusion).
```

### Analysis Performed, Negative Result
```
We appreciate this suggestion. We performed the requested analysis
and found [negative/null result]. We have added this to the Results
section and discuss its implications in the Discussion.

Changes made: Page X, lines Y-Z (results); Page A, lines B-C
(discussion).
```

## Literature Citation Templates

### Citation Added
```
We thank the reviewer for bringing this relevant work to our attention.
We have added [citation] and discuss its relevance to our findings.

Changes made: Page X, lines Y-Z.
```

### Multiple Citations Added
```
We appreciate the reviewer noting this gap in our literature coverage.
We have expanded our discussion to include [citations] and have revised
[section] accordingly.

Changes made: Page X, lines Y-Z; Page A, lines B-C.
```

### Citation Noted but Not Added
```
We thank the reviewer for this reference. While [citation] is certainly
relevant to [broader topic], we believe it is less directly applicable
to [our specific focus] because [reason]. We have, however, added
[alternative action if any].

Changes made: [None / Page X, lines Y-Z].
```

## Statistical Concern Templates

### Statistical Test Changed
```
We thank the reviewer for this statistical advice. We have re-analyzed
the data using [new method] as recommended. [Report new results.] Our
conclusions remain [unchanged / have been modified to reflect].

Changes made: Page X, lines Y-Z (methods); Page A, lines B-C (results).
```

### Statistical Justification Provided
```
We appreciate this query. We used [method] because [justification:
data properties, field conventions, etc.]. We have added text
explaining this rationale.

Changes made: Page X, lines Y-Z.
```

### Additional Statistical Information Provided
```
We thank the reviewer for requesting this information. We have added
[effect sizes / confidence intervals / sample sizes / etc.] throughout
the Results section.

Changes made: Page X, lines Y-Z; [additional locations].
```

```
We thank the editor for summarizing the reviewers' concerns and providing
guidance on priorities. We have addressed all points raised and have
particularly focused on [editor's highlighted priority].
```

### Addressing Editor's Additional Comment
```
Regarding the editor's comment on [topic]: We have [action taken].

Changes made: Page X, lines Y-Z.
```
