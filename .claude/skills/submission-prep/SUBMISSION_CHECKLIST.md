# Submission Checklist

Comprehensive checklist for manuscript submission.

## Before Starting Submission

### Journal Selection
- [ ] Confirmed scope matches your work
- [ ] Checked recent publications in the journal
- [ ] Reviewed author guidelines thoroughly
- [ ] Noted specific requirements (word limits, formats, etc.)
- [ ] Identified article type and any restrictions

### Co-Author Coordination
- [ ] All authors have reviewed final manuscript
- [ ] All authors have approved submission
- [ ] Author order confirmed
- [ ] Corresponding author designated
- [ ] Each author's contributions documented
- [ ] All ORCID IDs collected (if required)

---

## Manuscript Components

### Title Page
- [ ] Title within character/word limit
- [ ] All authors listed with correct names
- [ ] All affiliations complete and accurate
- [ ] Corresponding author identified with email
- [ ] Running title/short title (if required)
- [ ] Word count included (if required)
- [ ] Keywords included (correct number)

### Abstract
- [ ] Within word limit
- [ ] Correct format (structured vs. unstructured)
- [ ] All required headings present (if structured)
- [ ] No citations (unless journal allows)
- [ ] No undefined abbreviations
- [ ] Accurately reflects paper content

### Main Text

#### Introduction
- [ ] Clear statement of research question/objectives
- [ ] Appropriate background coverage
- [ ] Gap in knowledge identified
- [ ] Within any section-specific limits

#### Methods
- [ ] Sufficient detail for replication
- [ ] Established protocols cited
- [ ] Statistical methods described
- [ ] Software/tools with versions listed
- [ ] Ethics approval mentioned (if applicable)

#### Results
- [ ] Organized logically
- [ ] All figures/tables referenced
- [ ] Statistics reported correctly
- [ ] No interpretation (saved for Discussion)

#### Discussion
- [ ] Key findings summarized
- [ ] Results interpreted in context
- [ ] Limitations acknowledged
- [ ] Future directions suggested
- [ ] Conclusions appropriate to evidence

### References
- [ ] Format matches journal style
- [ ] All citations have matching references
- [ ] All references are cited in text
- [ ] DOIs included (if required)
- [ ] No broken links
- [ ] Recent literature included

### Supplementary Materials
- [ ] Clearly labeled (Fig. S1, Table S1, etc.)
- [ ] Referenced in main text
- [ ] Formatted according to guidelines
- [ ] File sizes within limits

---

## Figures

### Quality
- [ ] Resolution meets requirements (typically 300-600 DPI)
- [ ] Text readable at print size
- [ ] Scale bars included where needed
- [ ] Colors appropriate (accessible, consistent)
- [ ] No unnecessary decoration

### Formatting
- [ ] Correct file format (TIFF, EPS, PDF, etc.)
- [ ] Size within limits
- [ ] Correct color mode (RGB vs CMYK)
- [ ] Fonts embedded (if applicable)

### Labeling
- [ ] Panels labeled consistently (A, B, C...)
- [ ] Labels match figure legends
- [ ] Units included on axes
- [ ] All elements explained in legend

### Legends
- [ ] Clear, standalone descriptions
- [ ] All panels described
- [ ] Abbreviations defined
- [ ] Statistical information included
- [ ] Sample sizes noted

### Each Figure Checklist
- [ ] Figure 1: ___
- [ ] Figure 2: ___
- [ ] Figure 3: ___
- [ ] (Add rows as needed)

## Tables

### Content
- [ ] Clear, descriptive title
- [ ] Column headers with units
- [ ] Appropriate significant figures
- [ ] Consistent formatting
- [ ] Footnotes for abbreviations

### Formatting
- [ ] Correct file format
- [ ] No excessive formatting
- [ ] Readable font size
- [ ] Proper alignment

### Each Table Checklist
- [ ] Table 1: ___
- [ ] Table 2: ___
- [ ] (Add rows as needed)

## Required Statements

### Ethics and Consent
- [ ] IRB/Ethics approval number included
- [ ] Informed consent statement included
- [ ] Animal ethics approval (if applicable)
- [ ] Consent for publication (if applicable)

### Data and Code
- [ ] Data availability statement included
- [ ] Repository identified (if sharing)
- [ ] Accession numbers included (if applicable)
- [ ] Code availability statement (if applicable)

### Conflicts and Funding
- [ ] Conflict of interest disclosure
- [ ] Funding sources acknowledged
- [ ] Funder role described (if any)

### Author Contributions
- [ ] CRediT taxonomy used (if required)
- [ ] All authors included
- [ ] Contributions accurately described

---

## Cover Letter

- [ ] Correct editor name (if known)
- [ ] Manuscript title matches exactly
- [ ] Article type specified
- [ ] Key findings summarized
- [ ] Journal fit explained
- [ ] Required statements included
- [ ] Suggested reviewers (if requested)
- [ ] Excluded reviewers (if needed)
- [ ] Contact information complete
- [ ] All author names listed
- [ ] Proofread for errors

---

## File Preparation

### Naming Conventions
- [ ] Files named according to guidelines
- [ ] No special characters in filenames
- [ ] Version indicators clear

### File Types
- [ ] Manuscript in correct format (Word, LaTeX, PDF)
- [ ] Figures in correct format
- [ ] Tables in correct format
- [ ] Supplementary in correct format

### Organization
- [ ] All files accounted for
- [ ] Clear labeling
- [ ] Consistent organization

---

## Submission System

### Account Setup
- [ ] Account created/logged in
- [ ] Profile information current
- [ ] ORCID linked (if required)

### Metadata Entry
- [ ] Title entered correctly
- [ ] Abstract copied accurately
- [ ] Keywords entered
- [ ] All authors added with correct info
- [ ] Author order matches manuscript
- [ ] Corresponding author designated

### File Upload
- [ ] Manuscript uploaded
- [ ] Cover letter uploaded
- [ ] All figures uploaded in order
- [ ] Tables uploaded
- [ ] Supplementary files uploaded
- [ ] Response to reviewers (if revision)

### Review and Submit
- [ ] PDF proof reviewed carefully
- [ ] Figures display correctly in proof
- [ ] Tables formatted correctly in proof
- [ ] All files present in proof
- [ ] Final submission confirmed

---

## Post-Submission

- [ ] Confirmation email received
- [ ] Manuscript ID recorded
- [ ] All co-authors notified
- [ ] Files archived locally
- [ ] Calendar reminder for follow-up (if no response)

---

## Resubmission Additions

- [ ] Original manuscript ID referenced
- [ ] Response to reviewers complete
- [ ] Point-by-point format used
- [ ] All comments addressed
- [ ] Page/line numbers accurate
- [ ] Both clean and tracked versions ready
- [ ] Cover letter summarizes changes
- [ ] Any author changes noted
